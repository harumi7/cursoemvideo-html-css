Assuntos já estudados

Abaixo, deixo um resumo em lista de tudo que aprendi:

  • Paragráfos <p></p>, quebras <br>, emoji 😊 e símbolos ↑
  • Inserindo imagens simples <img src=linkdaimagemaqui.png alt=descriçãoimagem/>
  • Utilizando favicons
  • Hierarquia de títulos <h1></h1> ao <h6></h6>
  • Formatações de textos em HTML:
    1. destaque = utiliza-se a tag STRONG
    2. ênfase = tag EM
    3. texto marcado = tag MARK
    4. texto pequeno = tag SMALL
    5. texto deletado = tag DEL
    6. texto inserido = tag INS
    7. texto sobrescrito = tag SUP
    8. texto subscrito = tag SUB
  • Formatações de códigos, citações e abreviações.
    1. código-fonte = utiliza-se a tag <CODE> para deixar em fonte monospace e a tag PRE para deixar o texto estruturado 
              
num = int(input("Digite um número: "))
if num % 2 == 0:
    print(f"O número {num} é PAR.")
else:
    print(f"O número {num} é ÍMPAR.")
    2. citações simples: tag Q. ex: Como dizia minha amiga: Nem tudo que reluz é ouro.
    3. citações completas: tag BLOCKQUOTE CITE="LINKDAREFERÊNCIAAQUI". ex: Segundo Jairnilson Silva Paim, no seu livro O que é o SUS?:
      ...lembrando que o sistema de saúde é integrado não só pelos serviços de saúde, mas também pela mídia, escolas, financiadores, indústrias de equipamentos e de medicamentos, universidades, institutos de pesquisa etc.
    4. abreviações: tag ABBR. ex: Estou aprendendo HTML e Python. Depois de ter uma boa base em HTML, vou começar CSS.
    5. texto invertido: tag BDO DIR="DIREÇÃO". ex: A Bruxa Dourada Beatrice
  • Listas em HTML
    1. Lista ordenada: tag OL e tag LI em cada item
    2. Lista não ordenada: tag UL e tag LI em cada item
    3. Lista mista: tag OL e UL dentro uma da outra
    4. Lista de definição: tag DL; termo=tag DT, descrição=tag DD. ex:
      HTML
      Linguagem de marcação para a criação do conteúdo de um site.
      CSS
      Linguagem de marcação para a criação do design de um site.
      Javascript
      Linguagem de programação para a criação de interatividade de um site.
  • Links
    1. externos: <a href="linkaqui" target="_blank" ref="external"></a>
    2. internos: <a href="paginaaqui" rel="next">clique aqui</a>
    3. download: <a href="linkaqui.zip" download="linkaqui.zip" type="application/zip">clique aqui</a>
  • Imagens dinâmicas e áudio
  • Vídeos em HTML5
